# Hero-Mobile-Backend

## Setup

Create .env file in the root and add the following settings. Use your own AWS_SECRET_ACCESS_KEY and AWS_ACCESS_KEY_ID for local development and testing.

```.env
DB_HOST=
DB_PORT=
DB_USER=
DB_PASSWORD=
DB_NAME=
AWS_SECRET_ACCESS_KEY=
AWS_ACCESS_KEY_ID=
AWS_REGION=
S3_BUCKET_NAME=
COGNITO_USER_POOL=
COGNITO_CLIENT_ID=
```
